# Employee-Attendance-Analysis-SQL-and-PYTHON-project-


 Project Overview

This project focuses on analyzing employee attendance data using SQL. The goal is to answer real-world HR and operations questions such as attendance patterns, absenteeism, working hours, and consistency of employees.

This project is ideal for:

* Data Analyst interview preparation
* Practicing SQL queries (JOINs, GROUP BY, WHERE, DATE functions)
* Understanding real business scenarios using raw data
